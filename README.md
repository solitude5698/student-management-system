一、项目概述
本系统基于 Java Swing 开发桌面图形化界面，采用 DAO 分层架构，搭配 MySQL 数据库，区分管理员、教师、学生三类用户，覆盖班级、学生、教师、课程、选课、成绩、考勤全流程校园管理，适合课程设计与实训项目。
项目分层清晰：dao 数据访问层、model 实体模型层、util 通用工具层、view 界面视图层，通过 BaseDao 封装通用数据库操作，降低代码冗余，便于维护拓展。
二、核心功能模块
1. 用户登录与账号管理
支持管理员、教师、学生多角色登录；
提供统一修改密码功能，保障账号安全。
2. 基础信息管理（增删改查）
班级管理：新增、编辑、删除、查询班级信息；
学生管理：录入学生信息、批量管理、信息修改删除；
教师管理：教师档案新增、信息维护；
课程管理：课程录入、课程信息编辑与删除。
3. 学生选课业务
学生登录后可浏览可选课程，提交选课记录；系统校验重复选课、课程容量限制，保存学生 - 课程关联数据，支持选课记录查询与管理。
4. 成绩管理
教师 / 管理员录入学生对应课程成绩，支持成绩修改、删除、查看；可查看单学生全部课程分数，校验分数格式、重复录入等异常场景。
5. 考勤管理
考勤登记：按班级选择学生，标记出勤 / 缺勤 / 请假状态并记录日期；
考勤统计：按班级、时间段统计出勤率、缺勤率，可视化展示考勤数据，支持文件导出选择。
6. 数据统计分析
成绩统计：自动计算班级平均分等成绩指标；
考勤统计：批量核算出勤数据，生成统计结果。
三、技术架构亮点
分层解耦
dao 层统一处理数据库 CRUD，BaseDao 封装连接、事务、异常逻辑；model 实体映射数据库表；util 提供日期、字符串、数据库工具；view 层纯 Swing 界面，专注交互展示。
通用工具封装
内置数据库连接工具、日期格式化、字符串判空、文件选择工具，减少重复代码。
完善异常校验
选课、成绩录入、考勤操作均做业务校验，重复数据、格式错误给出界面提示。
